 进入补偿 DELETE 清理已上传数据

扫描并删除已上传云端且超过保留天数的本地采集目录。只删本地目录，不删数据库记录，支持 dry-run 预览和确认文本保护。

 进入清理 WebRTC 数据采集测试

实时预览 · WebRTC 多路传输。保留登录环境、任务 ID、状态、开始采集、停止采集和运行日志。

 进入采集 /api/monitor 系统监控面板

CPU 信息、内存信息、磁盘信息、平台信息和机器人状态，沿用原有字段与加载状态。

 进入系统 GET /debug/processes 进程监控面板

主进程及递归子进程指标。保留采样时间、读取快照、实时刷新、自动 5s 刷新与完整进程表列。

 进入进程 LOCK 采集锁解除

手动解锁采集流水线。保留当前状态字段、操作者、备注、解锁和刷新状态。

按记录 ID 检索 capture 元信息。保留概要、摄像头和 Segment 文件三个结果区域。
